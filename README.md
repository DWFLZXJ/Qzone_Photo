# 使用方法
* 安装Python 2.7.10；
* 安装依赖：pip install -r requirements.txt 
* 修改main.py的登录账户为你的qq账号和密码；
* 修改main.py的qq号为你需要的qq号；
* cmd中键入Python main.py运行。
